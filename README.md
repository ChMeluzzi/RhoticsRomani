# RhoticsRomani
Supporting material for the article Meli, G. &amp; Meluzzi, C. (submitted) "Rhotic variation in three Romani varieties: a synchronic and diachronic account", Studi AISV 15, Milano: Officinaventuno. 
It contains
- dataset of three Romani rhotics, with the different aggregate and non-aggregate variables
- R scripts for statistical analysis
- Figures and graphs created in R
- Coefficients tables and other supporting materials of the statistical analysis
